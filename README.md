# рҹ§  Emotion Detection & Learning Support Engine

An AI-powered learning assistant that detects learning-related emotions вҖ”
**Bored, Confident, Confused, Curious, Frustrated** вҖ” from a student's own
description of their problem, and responds with empathetic, field-aware
guidance generated by Google Gemini (with a built-in template fallback).

Built with a **dual-model pipeline** (BiLSTM + fine-tuned BERT) and a
**zero-dependency rule-based fallback**, so the application is always
runnable end-to-end вҖ” even before any model has been trained.

## вңЁ Key Features

- рҹҺҜ 5-class emotion detection from free-text academic problem descriptions
- рҹӨқ Mixed-emotion detection (e.g. *"Curious + Confused"*) at a вүҘ15% secondary-score threshold
- рҹ”Җ Side-by-side BiLSTM vs. BERT prediction comparison
- рҹ’¬ Gemini 2.5 Flash empathetic, field-specific responses with graceful template fallback
- рҹ“Ҡ Live analytics dashboard вҖ” emotion distribution, confidence timeline, field breakdown
- рҹӣҹ Fully functional out of the box via a rule-based fallback, even with no trained models present
- рҹ—„пёҸ CSV-based continuous-learning log, with an optional relational schema (`schema.sql`) for a hosted, multi-user future version

## рҹҡҖ Quick Start

```bash
git clone <your-repo-url>
cd 09_Programs_and_Codes
python -m venv venv && source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env        # optional: add your GOOGLE_API_KEY
streamlit run app.py
```

The app works immediately in fallback mode. To use the real trained
models, see [Training](#-training-the-models) below.

## рҹ§  Training the Models

```bash
cd 09_Programs_and_Codes/src
python train.py --model bilstm --data ../data/emotion_text_dataset.csv
python train.py --model bert   --data ../data/emotion_text_dataset.csv
```

This exports trained artifacts into `models/bltsm/` and
`models/bert_emotion_model_final/`. Restart the app to switch the sidebar
model-status badges from рҹҹЎ fallback to вң… loaded. See
[`models/bltsm/README.md`](09_Programs_and_Codes/models/bltsm/README.md)
and [`models/bert_emotion_model_final/README.md`](09_Programs_and_Codes/models/bert_emotion_model_final/README.md)
for the exact artifact list expected.
